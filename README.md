# 🐾 Panther

A glassmorphism OS-style UI with a built-in proxy browser powered by Ultraviolet.

## Setup

```bash
npm install
npm start
```

Then open: **http://localhost:8080**

## Browser

- Click the **Browse** icon in the taskbar to open the browser
- Type a URL or search term and press Enter
- Sites that allow embedding load directly in the window
- Sites that block iframes open in a new browser tab automatically
- With the server running, Ultraviolet handles proxying for most sites

## Features

- 🌊 Animated glassmorphism background
- 🕐 Live clock (center bar + taskbar)
- 🌐 Multi-tab proxy browser (Ultraviolet)
- ⚙️ Settings panel (particles, colors, clock format)
- 🖱️ Draggable & resizable browser window
- 🖥️ Fullscreen browser mode
